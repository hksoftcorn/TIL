# Git 복습

> 폴더를 생성하고 git을 생성합니다.

```git
$ git init
$ touch .gitignore
$ touch README.md
```

```git
$ git add .
$ git commit -m "pjt01 완료"
$ git remote add origin https://github.com/hksoftcorn/practice.git
$ git remote -v
$ git push -u origin master
```



- 원격 저장소 추가

```git
$ git remote add origin2 https://github.com/hksoftcorn/practice.git
$ git remote -v
$ git push -u origin2 master
```
